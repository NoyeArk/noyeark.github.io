概率论第7章：参数估计

1 参数估计

设有一个统计总体, 总体的分布函数为F(x,θ)F(x, \theta)F(x,θ)，其中θ\thetaθ为未知参数。X1,X2,⋯ ,XnX_{1}, X_{2}, \cdots, X_{n}X1​,X2​,⋯,Xn​是从总体X得到的样本，要根据该样本对参数θ\thetaθ做出估计，或估计θ\thetaθ的某个已知函数g(θ)g(\theta)g(θ)，这类问题成为参数估计。

已知总体的分布，来估计总体的参数。本章主要讲解参数估计中的点估计和区间估计。

2 点估计

2.1 点估计的概念

设总体X的分布函数F(x;θ)F(x ; \theta)F(x;θ)的形式已知，θ\thetaθ是待估参数，X1,X2,⋯ ,XnX_{1}, X_{2}, \cdots, X_{n}X1​,X2​,⋯,Xn​是X的一个样本，x1,x2,⋯ ,xnx_{1}, x_{2}, \cdots, x_{n}x1​,x2​,⋯,xn​是相应的样本值，用样本值估计参数值。

定义：构造一个适当的估计量θ^(X1,X2,⋯ ,Xn)\hat{\theta}\left(X_{1}, X_{2}, \cdots, X_{n}\right)θ^(X1​,X2​,⋯,Xn​)，用它的观察值θ^(x1,x2,⋯ ,xn)\hat{\theta}\left(x_{1}, x_{2}, \cdots, x_{n}\right)θ^(x1​,x2​,⋯,xn​)来估计未知参数θ\thetaθ，称θ^(x1,x2,⋯ ,xn)\hat{\theta}\left(x_{1}, x_{2}, \cdots, x_{n}\right)θ^(x1​,x2​,⋯,xn​)为θ\thetaθ的估计量。这样的估计称为点估计。

寻求估计量的常用方法：

  1. 矩估计法
  2. 最大似然法
  3. 最小二乘法
  4. 贝叶斯方法

2.2 矩估计法

2.2.1 定义

矩估计法是英国统计学家K.皮尔逊最早提出的。该法以大数定律为理论依据，用样本矩估计总体矩。

定义：若 X为连续型随机变量其概率密度为

f(x;θ1,θ2,⋯ ,θk)f\left(x ; \theta_{1}, \theta_{2}, \cdots, \theta_{k}\right) f(x;θ1​,θ2​,⋯,θk​)

若X为离散型随机变量，其分布律为

P{X=x}=p(x;θ1,θ2,⋯ ,θk)P\{X=x\}=p\left(x ; \theta_{1}, \theta_{2}, \cdots, \theta_{k}\right) P{X=x}=p(x;θ1​,θ2​,⋯,θk​)

其中θ1,θ2,⋯ ,θk\theta_{1}, \theta_{2}, \cdots, \theta_{k}θ1​,θ2​,⋯,θk​为待估参数，X1,X2,⋯ ,XnX_{1}, X_{2}, \cdots, X_{n}X1​,X2​,⋯,Xn​是来自总体X的样本。

假设总体X的前k阶矩存在，即

μl=E(Xl)=∫−∞+∞xlf(x;θ1,θ2,⋯ ,θk)dxX连续型\mu_{l}=E\left(X^{l}\right)=\int_{-\infty}^{+\infty} x^{l} f\left(x ; \theta_{1}, \theta_{2}, \cdots, \theta_{k}\right) d x \quad X 连续型 μl​=E(Xl)=∫−∞+∞​xlf(x;θ1​,θ2​,⋯,θk​)dxX连续型

μl=E(Xl)=∑xlp(x;θ1,θ2,⋯ ,θk)X离散型\mu_{l}=E\left(X^{l}\right)=\sum x^{l} p\left(x ; \theta_{1}, \theta_{2}, \cdots, \theta_{k}\right) \quad X 离散型 μl​=E(Xl)=∑xlp(x;θ1​,θ2​,⋯,θk​)X离散型

用样本原点矩估计相应的总体原点矩，又用样本原点矩的连续函数估计相应的总体原点矩的连续函数，这种参数点估计法称为矩估计法。

那么用各μi\mu_{i}μi​的估计量AiA_{i}Ai​分别代替上式中的诸μi\mu_{i}μi​，即可得到诸θj{\theta}_{j}θj​的矩估计量：

θ^j=θj(A1,A2,⋯ ,Ak)j=1,2,⋯ ,k\hat{\theta}_j=\theta_j(A_1,A_2,\cdots,A_k)\quad j=1,2,\cdots,k θ^j​=θj​(A1​,A2​,⋯,Ak​)j=1,2,⋯,k

2.3 最大似然估计法

它是在总体类型已知条件下使用的一种参数估计方法，它首先是由德国数学家高斯在1821年提出的，最大似然估计法，是建立在最大似然原理的基础上的求点估计量的方法。

2.3.1 最大似然估计原理

若X为连续型随机变量，密度函数为f(x;θ),θ∈Θf(x ; \theta), \theta \in \Thetaf(x;θ),θ∈Θ，当给定样本X1,X2,…,XnX_{1}, X_{2}, \ldots, X_{n}X1​,X2​,…,Xn​时，定义似然函数为：

L(θ)=L(x1,x2,…,xn;θ)=∏i=1nf(xi;θ)L(\theta)=L\left(x_{1}, x_{2}, \ldots, x_{n} ; \theta\right)=\prod_{i=1}^{n} f\left(x_{i} ; \theta \right) L(θ)=L(x1​,x2​,…,xn​;θ)=i=1∏n​f(xi​;θ)

若L(θ^)=max⁡θL(θ)L(\hat{\theta})=\max _{\theta} L(\theta)L(θ^)=maxθ​L(θ)，则有以下：

  • θ^(x1,x2,…,xn)称为θ的最大似然估计值\hat{\theta}\left(x_{1}, x_{2}, \ldots, x_{n}\right) 称为 \theta 的最大似然估计值θ^(x1​,x2​,…,xn​)称为θ的最大似然估计值
  • θ^(X1,…,Xn)称为θ的最大似然估计量\hat{\theta}\left(X_{1}, \ldots, X_{n}\right) 称为 \theta 的最大似然估计量θ^(X1​,…,Xn​)称为θ的最大似然估计量

最大似然原理的直观想法是：在试验中概率最大的事件最有可能出现。

2.3.2 最大似然估计量的一般步骤

（1）定义似然函数L(θ) L(\theta)L(θ)

（2）一般地，求出ln⁡L(θ)\ln L(\theta)lnL(θ)及似然方程

∂ln⁡L(θ)∂θi∣θ=θ^=0(i=1,2,…,m)\left.\frac{\partial \ln L(\theta)}{\partial \theta_{i}}\right|_{\theta=\hat{\theta}}=0 \quad(i={1 , 2}, \ldots, m) ∂θi​∂lnL(θ)​∣∣∣∣∣​θ=θ^​=0(i=1,2,…,m)

（3）解似然方程得到最大似然估计值

θ^i=θ^i(x1,x2,…,xn)(i=1,2,…,m)\hat{\theta}_i=\hat{\theta}_i\left(x_1,x_2,\ldots,x_n\right)\quad(i={1},{2},\ldots,m) θ^i​=θ^i​(x1​,x2​,…,xn​)(i=1,2,…,m)

2.3.3 最大似然估计的不变性

若θ\thetaθ估计值为θ^\hat{\theta}θ^，则u=u(θ)u=u(\theta)u=u(θ)估计值为u^=u(θ^)\hat{u}=u(\hat{\theta})u^=u(θ^)，条件是u=u(θ)u=u(\theta)u=u(θ)具有单值反函数θ=θ(u)\theta=\theta(u)θ=θ(u)。

3 估计量的评选标准

3.1 无偏性

若估计量θ^=θ(X1,X2,⋯ ,Xn)\hat{\theta}=\theta\left(X_{1}, X_{2}, \cdots, X_{n}\right)θ^=θ(X1​,X2​,⋯,Xn​)的数学期望E(θ^)E(\hat{\theta})E(θ^)对任意的θ∈Θ\theta \in \Thetaθ∈Θ，都有E(θ^)=θE(\hat{\theta})=\thetaE(θ^)=θ，则称θ^\hat{\theta}θ^是θ\thetaθ的无偏估计。无偏性就是求期望。

D(θ^1)≤D(θ^2)D(\hat{\theta}_1) \leq D(\hat{\theta}_2) D(θ^1​)≤D(θ^2​)

3.3 相合性

前面讲的无偏性和有效性都是在样本容量n固定的前提下提出的。希望随着样本容量的增大，一个估计量的值稳定于待估参数的真值。这样，对估计量有以下相合性的要求。

4 区间估计

4.1 置信区间定义

设θ\thetaθ是一个待估参数，给定α>0，若由样本X1,X2,…,XnX_{1}, X_{2}, \ldots, X_{n}X1​,X2​,…,Xn​确定的两个统计量θ‾=θ‾(X1,X2,⋯ ,Xn)\underline{\theta}=\underline{\theta}\left(X_{1}, X_{2}, \cdots, X_{n}\right)θ​=θ​(X1​,X2​,⋯,Xn​)和θˉ=θˉ(X1,X2,⋯ ,Xn)\bar{\theta}=\bar{\theta}\left(X_{1}, X_{2}, \cdots, X_{n}\right)θˉ=θˉ(X1​,X2​,⋯,Xn​)，其中θ‾<θˉ\underline{\theta}<\bar{\theta}θ​<θˉ，满足：

P{θ‾<θ<θˉ}≥1−αP\{\underline{\theta}<\theta<\bar{\theta}\} \geq 1-\alpha P{θ​<θ<θˉ}≥1−α

则称区间(θ‾,θˉ)(\underline{\theta}, \bar{\theta})(θ​,θˉ)是θ\thetaθ的置信水平（置信度）为1-α的置信区间。其中，$\underline{\theta} 和 \bar{\theta} $分别称为置信下限和置信上限。

4.2 置信区间的求法

（1）寻找参数θ\thetaθ的一个良好的点估计T(X1,X2,…Xn)T\left(X_{1}, X_{2}, \ldots X_{n}\right)T(X1​,X2​,…Xn​)

（2）寻找一个待估参数θ\thetaθ和估计量T的函数U(T,θ)U(T, \theta)U(T,θ)，且其分布为已知

（3）对于给定的置信水平1-α，根据U(T,θ)U(T, \theta)U(T,θ)的分布，确定常熟a, b，使得

P(a<U(T,θ)<b)=1−α\boldsymbol{P}(\boldsymbol{a}<\boldsymbol{U}(T, \boldsymbol{\theta})<\boldsymbol{b})=1-\alpha P(a<U(T,θ)<b)=1−α

解得P{θ‾<θ<θˉ}=1−αP\{\underline{\theta}<\theta<\bar{\theta}\}=1-\alphaP{θ​<θ<θˉ}=1−α。
