nдёӘдёҚеҗҢе…ғзҙ иҝӣж ҲпјҢеҮәж Ҳе…ғзҙ дёҚеҗҢжҺ’еҲ—зҡ„дёӘж•°жҳҜ $ \frac{1}{n + 1} C_{2n}^{n} $пјҢдёҠиҝ°е…¬ејҸиў«з§°дёәеҚЎзү№е…°пјҲCatalanпјүж•°гҖӮ

1.2 ж Ҳзҡ„еӯҳеӮЁз»“жһ„

1.2.1 йЎәеәҸеӯҳеӮЁ

йҮҮз”ЁйЎәеәҸеӯҳеӮЁзҡ„ж Ҳз§°дёәйЎәеәҸж ҲпјҢе®ғеҲ©з”ЁдёҖз»„ең°еқҖиҝһз»ӯзҡ„еӯҳеӮЁеҚ•е…ғеӯҳж”ҫиҮӘж Ҳеә•еҲ°ж ҲйЎ¶зҡ„ж•°жҚ®е…ғзҙ пјҢеҗҢж—¶йҷ„и®ҫдёҖдёӘжҢҮй’ҲпјҲtopпјүжҢҮзӨәеҪ“еүҚж ҲйЎ¶е…ғзҙ зҡ„дҪҚзҪ®гҖӮ

иӢҘзҺ°еңЁжңүдёҖдёӘж ҲпјҢ$StackSize=5$пјҢеҲҷж Ҳзҡ„жҷ®йҖҡжғ…еҶөгҖҒз©әж ҲгҖҒж»Ўж Ҳзҡ„жғ…еҶөеҲҶеҲ«еҰӮдёӢеӣҫжүҖзӨәпјҡ

1.2.2 е…ұдә«ж Ҳ

еҲ©з”Ёж Ҳеә•дҪҚзҪ®зӣёеҜ№дёҚеҸҳзҡ„зү№еҫҒпјҢеҸҜи®©дёӨдёӘйЎәеәҸж Ҳе…ұдә«дёҖдёӘдёҖз»ҙж•°з»„з©әй—ҙпјҢе°ҶдёӨдёӘж Ҳзҡ„ж Ҳеә•еҲҶеҲ«и®ҫзҪ®еңЁе…ұдә«з©әй—ҙзҡ„дёӨз«ҜпјҢдёӨдёӘж ҲйЎ¶еҗ‘е…ұдә«з©әй—ҙзҡ„дёӯй—ҙе»¶дјёпјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ

дёӨдёӘж Ҳзҡ„ж ҲйЎ¶жҢҮй’ҲйғҪжҢҮеҗ‘ж ҲйЎ¶е…ғзҙ пјҢ$ top_0 = -1 $ ж—¶ 0 еҸ·ж Ҳдёәз©әпјҢ$ top_1 = MaxSize $ ж—¶ 1 еҸ·ж Ҳдёәз©әпјӣ

д»…еҪ“дёӨдёӘж ҲйЎ¶жҢҮй’ҲзӣёйӮ» $ top_0 + 1 = top_1 $ ж—¶пјҢеҲӨж–ӯдёәж Ҳж»ЎгҖӮеҪ“ 0 еҸ·ж Ҳиҝӣж Ҳж—¶ $ top_0 $ е…ҲеҠ  1 еҶҚиөӢеҖјпјҢ1 еҸ·ж Ҳиҝӣж Ҳж—¶ $ top_1 $ е…ҲеҮҸдёҖеҶҚиөӢеҖјеҮәж Ҳж—¶еҲҷеҲҡеҘҪзӣёеҸҚгҖӮ

1.2.3 й“ҫејҸеӯҳеӮЁз»“жһ„

йҮҮз”Ёй“ҫејҸеӯҳеӮЁзҡ„ж Ҳз§°дёәй“ҫж ҲпјҢй“ҫж Ҳзҡ„дјҳзӮ№жҳҜдҫҝдәҺеӨҡдёӘж Ҳе…ұдә«еӯҳеӮЁз©әй—ҙе’ҢжҸҗй«ҳе…¶ж•ҲзҺҮпјҢдё”дёҚеӯҳеңЁж Ҳж»ЎдёҠжәўзҡ„жғ…еҶөгҖӮйҖҡеёёйҮҮз”ЁеҚ•й“ҫиЎЁе®һзҺ°пјҢе№¶и§„е®ҡжүҖжңүж“ҚдҪңйғҪжҳҜеңЁеҚ•й“ҫиЎЁзҡ„иЎЁеӨҙиҝӣиЎҢзҡ„гҖӮ

иҝҷйҮҢи§„е®ҡй“ҫж ҲжІЎжңүеӨҙиҠӮзӮ№пјҢLheadжҢҮеҗ‘ж ҲйЎ¶е…ғзҙ пјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

еҜ№дәҺз©әж ҲжқҘиҜҙпјҢй“ҫиЎЁеҺҹе®ҡд№үжҳҜеӨҙжҢҮй’ҲжҢҮеҗ‘з©әпјҢйӮЈд№Ҳй“ҫж Ҳзҡ„з©әе…¶е®һе°ұжҳҜ $ top = nullptr$ зҡ„ж—¶еҖҷгҖӮ

1.2.4 жҖ§иғҪеҲҶжһҗ

й“ҫж Ҳзҡ„иҝӣж Ҳpushе’ҢеҮәж Ҳpopж“ҚдҪңйғҪеҫҲз®ҖеҚ•пјҢж—¶й—ҙеӨҚжқӮеәҰеқҮдёә$O(1)$гҖӮ

  • еҜ№жҜ”дёҖдёӢйЎәеәҸж ҲдёҺй“ҫж ҲпјҢе®ғд»¬еңЁж—¶й—ҙеӨҚжқӮеәҰдёҠжҳҜдёҖж ·зҡ„пјҢеқҮдёә$O(1)$гҖӮ
  • еҜ№дәҺз©әй—ҙжҖ§иғҪпјҡ
    • йЎәеәҸж ҲйңҖиҰҒдәӢе…ҲзЎ®е®ҡдёҖдёӘеӣәе®ҡзҡ„й•ҝеәҰпјҢеҸҜиғҪдјҡеӯҳеңЁеҶ…еӯҳз©әй—ҙжөӘиҙ№зҡ„й—®йўҳпјҢдҪҶе®ғзҡ„дјҳеҠҝжҳҜеӯҳеҸ–ж—¶е®ҡдҪҚеҫҲж–№дҫҝгҖӮ
    • й“ҫж ҲеҲҷиҰҒжұӮжҜҸдёӘе…ғзҙ йғҪжңүжҢҮй’ҲеҹҹпјҢиҝҷеҗҢж—¶д№ҹеўһеҠ дәҶдёҖдәӣеҶ…еӯҳејҖй”ҖпјҢдҪҶеҜ№дәҺж Ҳзҡ„й•ҝеәҰж— йҷҗеҲ¶гҖӮ
  • жүҖд»Ҙе®ғд»¬зҡ„еҢәеҲ«е’ҢзәҝжҖ§иЎЁдёӯи®Ёи®әзҡ„дёҖж ·пјҢеҰӮжһңж Ҳзҡ„дҪҝз”ЁиҝҮзЁӢдёӯе…ғзҙ еҸҳеҢ–дёҚеҸҜйў„ж–ҷпјҢжңүж—¶еҫҲе°ҸпјҢжңүж—¶йқһеёёеӨ§пјҢйӮЈд№ҲжңҖеҘҪжҳҜз”Ёй“ҫж ҲгҖӮ
  • еҸҚд№ӢпјҢеҰӮжһңе®ғзҡ„еҸҳеҢ–еңЁеҸҜжҺ§иҢғеӣҙеҶ…пјҢе»әи®®дҪҝз”ЁйЎәеәҸж ҲдјҡжӣҙеҘҪдёҖдәӣгҖӮ

1.3 ж Ҳзҡ„еә”з”Ё

1.3.1 йҖ’еҪ’

йҖ’еҪ’жҳҜдёҖз§ҚйҮҚиҰҒзҡ„зЁӢеәҸи®ҫи®Ўж–№жі•гҖӮз®ҖеҚ•ең°иҜҙпјҢиӢҘеңЁдёҖдёӘеҮҪж•°гҖҒиҝҮзЁӢжҲ–ж•°жҚ®з»“жһ„зҡ„е®ҡд№үдёӯеҸҲеә”з”ЁдәҶе®ғиҮӘиә«пјҢеҲҷиҝҷдёӘеҮҪж•°гҖҒиҝҮзЁӢжҲ–ж•°жҚ®з»“жһ„з§°дёәжҳҜйҖ’еҪ’е®ҡд№үзҡ„пјҢз®Җз§°йҖ’еҪ’гҖӮ

е®ғйҖҡеёёжҠҠдёҖдёӘеӨ§еһӢзҡ„еӨҚжқӮй—®йўҳеұӮеұӮиҪ¬еҢ–дёәдёҖдёӘдёҺеҺҹй—®йўҳзӣёдјјзҡ„и§„жЁЎиҫғе°Ҹзҡ„й—®йўҳжқҘжұӮи§ЈгҖӮйҖ’еҪ’зӯ–з•ҘеҸӘйңҖе°‘йҮҸзҡ„д»Јз Ғе°ұеҸҜд»ҘжҸҸиҝ°еІҖи§ЈйўҳиҝҮзЁӢжүҖйңҖиҰҒзҡ„еӨҡж¬ЎйҮҚеӨҚи®Ўз®—пјҢеӨ§еӨ§еҮҸе°‘дәҶзЁӢеәҸзҡ„д»Јз ҒйҮҸдҪҶеңЁйҖҡеёёжғ…еҶөдёӢпјҢе®ғзҡ„ж•ҲзҺҮе№¶дёҚжҳҜеӨӘй«ҳгҖӮ

1.3.2 жӢ¬еҸ·еҢ№й…Қ

1.3.3 иЎЁиҫҫејҸжұӮеҖј

1.3.3.1 еҗҺзјҖиЎЁиҫҫејҸи®Ўз®—з»“жһң

иЎЁиҫҫејҸжұӮеҖјжҳҜзЁӢеәҸи®ҫи®ЎиҜӯиЁҖзј–иҜ‘дёӯдёҖдёӘжңҖеҹәжң¬зҡ„й—®йўҳпјҢе®ғзҡ„е®һзҺ°жҳҜж Ҳеә”з”Ёзҡ„дёҖдёӘе…ёеһӢиҢғдҫӢгҖӮ

  • дёӯзјҖиЎЁиҫҫејҸдёҚд»…дҫқиө–иҝҗз®—з¬Ұзҡ„дјҳе…Ҳзә§пјҢиҖҢдё”иҝҳиҰҒеӨ„зҗҶжӢ¬еҸ·гҖӮ
  • еҗҺзјҖиЎЁиҫҫејҸзҡ„иҝҗз®—з¬ҰеңЁж“ҚдҪңж•°еҗҺйқўпјҢеңЁеҗҺзјҖиЎЁиҫҫејҸдёӯе·ІиҖғиҷ‘дәҶиҝҗз®—з¬Ұзҡ„дјҳе…Ҳзә§пјҢжІЎжңүжӢ¬еҸ·пјҢеҸӘжңүж“ҚдҪңж•°е’Ңиҝҗз®—з¬ҰгҖӮ

дҫӢеҰӮдёӯзјҖиЎЁиҫҫејҸ $ A + B вҲ— ( C вҲ’ D ) вҲ’ E / F $ жүҖеҜ№еә”зҡ„еҗҺзјҖиЎЁиҫҫејҸдёә $ A B C D вҲ’ вҲ— + E F / $гҖӮ

еҗҺзјҖиЎЁиҫҫејҸи®Ўз®—и§„еҲҷпјҡд»Һе·ҰеҲ°еҸійҒҚеҺҶиЎЁиҫҫејҸзҡ„жҜҸдёӘж•°еӯ—е’Ңз¬ҰеҸ·пјҢйҒҮеҲ°жҳҜж•°еӯ—е°ұиҝӣж ҲпјҢйҒҮеҲ°жҳҜз¬ҰеҸ·пјҢе°ұе°ҶеӨ„дәҺж ҲйЎ¶дёӨдёӘж•°еӯ—еҮәж ҲпјҢиҝӣйЎ№иҝҗз®—пјҢиҝҗз®—з»“жһңиҝӣж ҲпјҢдёҖзӣҙеҲ°жңҖз»ҲиҺ·еҫ—з»“жһңгҖӮ

еҗҺзјҖиЎЁиҫҫејҸ $ A B C D вҲ’ вҲ— + E F / вҲ’ $ жұӮеҖјзҡ„иҝҮзЁӢйңҖиҰҒ12жӯҘпјҢеҰӮдёӢиЎЁжүҖзӨәпјҡ

иҜ»иҖ…д№ҹеҸҜе°ҶеҗҺзјҖиЎЁиҫҫејҸдёҺеҺҹиҝҗз®—ејҸеҜ№еә”зҡ„иЎЁиҫҫејҸж ‘пјҲз”ЁжқҘиЎЁзӨәз®—жңҜиЎЁиҫҫејҸзҡ„дәҢе…ғж ‘пјүзҡ„еҗҺеәҸйҒҚеҺҶиҝӣиЎҢжҜ”иҫғ,еҸҜд»ҘеҸ‘зҺ°е®ғд»¬жңүејӮжӣІеҗҢе·Ҙд№ӢеҰҷгҖӮ

еҰӮдёӢеӣҫеҲҷжҳҜ $ A + B вҲ— ( C вҲ’ D ) вҲ’ E / F $ еҜ№еә”зҡ„иЎЁиҫҫејҸпјҢе®ғзҡ„еҗҺеәҸйҒҚеҺҶпјҲе·Ұеӯҗж ‘вҶ’еҸіеӯҗж ‘вҶ’ж №пјүеҚіжҳҜиЎЁиҫҫејҸ $ A B C D вҲ’ вҲ— + E F / $гҖӮ

1.3.3.2 дёӯзјҖиЎЁиҫҫејҸиҪ¬еҗҺзјҖиЎЁиҫҫејҸ

жҠҠе№іж—¶жүҖз”Ёзҡ„ж ҮеҮҶеӣӣеҲҷиҝҗз®—иЎЁиҫҫејҸпјҢеҚі $ a + b вҲ’ a вҲ— ( ( c + d ) / e вҲ’ f ) + g $ еҸ«еҒҡдёӯзјҖиЎЁиҫҫејҸгҖӮ

еӣ дёәжүҖжңүзҡ„иҝҗз®—з¬ҰеҸ·йғҪеңЁдёӨж•°еӯ—зҡ„дёӯй—ҙпјҢзҺ°еңЁй—®йўҳе°ұжҳҜдёӯзјҖеҲ°еҗҺзјҖзҡ„иҪ¬еҢ–гҖӮ

и§„еҲҷпјҡд»Һе·ҰеҲ°еҸійҒҚеҺҶдёӯзјҖиЎЁиҫҫејҸзҡ„жҜҸдёӘж•°еӯ—е’Ңз¬ҰеҸ·гҖӮиӢҘжҳҜж•°еӯ—е°ұиҫ“еҮәпјҢеҚіжҲҗдёәеҗҺзјҖиЎЁиҫҫејҸзҡ„дёҖйғЁеҲҶпјӣиӢҘжҳҜз¬ҰеҸ·пјҢеҲҷеҲӨж–ӯе…¶дёҺж ҲйЎ¶з¬ҰеҸ·зҡ„дјҳе…Ҳзә§пјҢжҳҜеҸіжӢ¬еҸ·жҲ–дјҳе…Ҳзә§дҪҺдәҺж ҲйЎ¶з¬ҰеҸ·пјҲд№ҳйҷӨдјҳе…ҲеҠ еҮҸпјүеҲҷж ҲйЎ¶е…ғзҙ дҫқж¬ЎеҮәж Ҳе№¶иҫ“еҮәпјҢе№¶е°ҶеҪ“еүҚз¬ҰеҸ·иҝӣж ҲпјҢдёҖзӣҙеҲ°жңҖз»Ҳиҫ“еҮәеҗҺзјҖиЎЁиҫҫејҸдёәжӯўгҖӮ

дҫӢпјҡе°ҶдёӯзјҖиЎЁиҫҫејҸ $ a + b вҲ’ a вҲ— ( ( c + d ) / e вҲ’ f ) + g $ иҪ¬еҢ–дёәзӣёеә”зҡ„еҗҺзјҖиЎЁиҫҫејҸгҖӮ

еҲҶжһҗпјҡйңҖиҰҒж №жҚ®ж“ҚдҪңз¬Ұзҡ„дјҳе…Ҳзә§жқҘиҝӣиЎҢж Ҳзҡ„еҸҳеҢ–пјҢжҲ‘д»¬з”ЁicpжқҘиЎЁзӨәеҪ“еүҚжү«жҸҸеҲ°зҡ„иҝҗз®—з¬Ұchзҡ„дјҳе…Ҳзә§пјҢиҜҘиҝҗз®—з¬Ұиҝӣж ҲеҗҺзҡ„дјҳе…Ҳзә§дёәispпјҢеҲҷиҝҗз®—з¬Ұзҡ„дјҳе…Ҳзә§еҰӮдёӢиЎЁжүҖзӨәпјҢе…¶дёӯispжҳҜж ҲеҶ…дјҳе…ҲпјҲin stack priorityпјүж•°пјҢicpжҳҜж ҲеӨ–дјҳе…ҲпјҲin coming priorityпјүж•°гҖӮ

жҲ‘д»¬еңЁиЎЁиҫҫејҸеҗҺйқўеҠ дёҠз¬ҰеҸ·вҖҳ#вҖҷпјҢиЎЁзӨәиЎЁиҫҫејҸз»“жқҹгҖӮе…·дҪ“иҪ¬жҚўиҝҮзЁӢеҰӮдёӢпјҡ

еҚізӣёеә”зҡ„еҗҺзјҖиЎЁиҫҫејҸдёә $ a b + a c d + e / f вҲ’ вҲ— вҲ’ g + $гҖӮ

2 йҳҹеҲ—

2.1 йҳҹеҲ—зҡ„еҹәжң¬жҰӮеҝө

йҳҹеҲ—пјҡдёҖз§Қе…Ҳиҝӣе…ҲеҮәзҡ„зәҝеҪўиЎЁгҖӮеҸӘе…Ғи®ёеңЁиЎЁдёҖз«ҜжҸ’е…ҘпјҢеңЁеҸҰдёҖз«ҜеҲ йҷӨгҖӮ

  • йҳҹе°ҫrearпјҡжҸ’е…Ҙз«ҜпјҢзәҝжҖ§иЎЁзҡ„иЎЁе°ҫгҖӮ
  • йҳҹеӨҙfrontпјҡеҲ йҷӨз«ҜпјҢзәҝжҖ§иЎЁзҡ„иЎЁеӨҙгҖӮ

йҳҹеҲ—зҡ„иҝӣйҳҹе’ҢеҮәйҳҹпјҡ

2.2 йҳҹеҲ—зҡ„йЎәеәҸеӯҳеӮЁз»“жһ„

2.2.1 йҳҹеҲ—зҡ„йЎәеәҸеӯҳеӮЁ

йЎәеәҸеӯҳеӮЁжҳҜжҢҮеҲҶй…ҚдёҖеқ—иҝһз»ӯзҡ„еӯҳеӮЁеҚ•е…ғеӯҳж”ҫйҳҹеҲ—дёӯзҡ„е…ғзҙ пјҢе№¶йҷ„и®ҫдёӨдёӘжҢҮй’ҲпјҡйҳҹеӨҙжҢҮй’Ҳе’Ңйҳҹе°ҫжҢҮй’ҲгҖӮ

д»Јз ҒеҰӮдёӢпјҡ

еҰӮдёӢеӣҫжүҖзӨәпјҢжӯӨж—¶е…ҘйҳҹеҮәзҺ°вҖңдёҠжәўеҮәвҖқпјҢдҪҶиҝҷз§ҚжәўеҮәдёҚжҳҜзңҹжӯЈзҡ„жәўеҮәпјҢеңЁdataж•°з»„дёӯдҫқз„¶еӯҳеңЁеҸҜд»Ҙеӯҳж”ҫе…ғзҙ зҡ„з©әдҪҚзҪ®пјҢжүҖд»ҘжҳҜдёҖз§ҚеҒҮжәўеҮәгҖӮ

2.2.2 еҫӘзҺҜйҳҹеҲ—

е°ҶйЎәеәҸйҳҹеҲ—иҮҶйҖ жҲҗдёҖдёӘзҺҜзҠ¶зҡ„з©әй—ҙпјҢеҚіжҠҠеӯҳеӮЁйҳҹеҲ—е…ғзҙ зҡ„иЎЁд»ҺйҖ»иҫ‘дёҠи§ҶдёәдёҖдёӘзҺҜпјҢз§°дёәеҫӘзҺҜйҳҹеҲ—гҖӮиӢҘrear+1==MпјҢеҲҷд»Өrear=0;

  • е…Ҙйҳҹпјҡrear=(rear+1) % M; sq[rear]=x;
  • еҮәйҳҹпјҡfront=(front+1) % M; x = sq[front];

дёәдәҶеҢәеҲҶйҳҹз©әиҝҳжҳҜйҳҹж»ЎпјҢжңүдёүз§ҚеӨ„зҗҶж–№ејҸпјҡ

  1. еҸҰеӨ–и®ҫдёҖдёӘж Үеҝ—д»ҘеҢәеҲ«йҳҹз©әгҖҒйҳҹж»Ў
  2. зұ»еһӢдёӯеўһи®ҫиЎЁзӨәе…ғзҙ дёӘж•°зҡ„ж•°жҚ®жҲҗе‘ҳпјҢиҝҷж ·йҳҹз©әзҡ„жқЎд»¶дёә$Q.size = 0$пјҢйҳҹж»Ўзҡ„жқЎд»¶дёә$Q.front = Q.rear$
  3. зүәзүІдёҖдёӘеҚ•е…ғжқҘеҢәеҲҶйҳҹз©әе’Ңйҳҹж»ЎпјҢе…Ҙйҳҹж—¶е°‘з”ЁдёҖдёӘйҳҹеҲ—еҚ•е…ғпјҢиҝҷжҳҜдёҖз§Қиҫғдёәжҷ®йҒҚзҡ„еҒҡжі•гҖӮ
    • йҳҹз©әпјҡ$Q.front = Q.rear$
    • йҳҹж»Ўпјҡ$(Q.rear + 1) % M = Q.front$

2.3 йҳҹеҲ—зҡ„й“ҫејҸеӯҳеӮЁз»“жһ„

2.3.1 йҳҹеҲ—зҡ„й“ҫејҸеӯҳеӮЁ

йҳҹеҲ—зҡ„й“ҫејҸиЎЁзӨәз§°дёәй“ҫйҳҹеҲ—пјҢе®ғе®һйҷ…дёҠжҳҜдёҖдёӘеҗҢж—¶еёҰжңүйҳҹеӨҙжҢҮй’Ҳе’Ңйҳҹе°ҫжҢҮй’Ҳзҡ„еҚ•й“ҫиЎЁгҖӮеӨҙжҢҮй’ҲжҢҮеҗ‘йҳҹеӨҙз»“зӮ№пјҢе°ҫжҢҮй’ҲжҢҮеҗ‘йҳҹе°ҫз»“зӮ№пјҢеҚіеҚ•й“ҫиЎЁзҡ„жңҖеҗҺдёҖдёӘз»“зӮ№гҖӮ

д»Јз ҒеҰӮдёӢпјҡ

  • еҪ“$Q.front == null$ дё” $Q.rear == null$ж—¶пјҢй“ҫйҳҹеҲ—дёәз©әгҖӮ

дёҚеёҰеӨҙиҠӮзӮ№зҡ„й“ҫйҳҹеҲ—еңЁж“ҚдҪңдёҠеҫҖеҫҖжҜ”иҫғйә»зғҰпјҢеӣ жӯӨйҖҡеёёе°Ҷй“ҫйҳҹеҲ—и®ҫи®ЎжҲҗдёҖдёӘеёҰеӨҙиҠӮзӮ№зҡ„еҚ•й“ҫиЎЁгҖӮ
